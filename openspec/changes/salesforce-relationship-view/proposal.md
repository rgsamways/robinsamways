## Why

The shipped `salesforce-loan-demo` proves the integration is real, but it's a flat table — it doesn't yet demonstrate the two things most specific to why Servus is hiring for this role: unifying relationships across records (their "Relationship Empowerment Platforms Team"), and AI-assisted workflow (their Agentforce/Deloitte pilot). Extending the demo to speak directly to both, honestly and without overclaiming licensed products Robin doesn't have, makes this piece noticeably more targeted to the actual interview than a generic "I can call a REST API" showcase.

## What Changes

- Add a per-Account relationship view: select an Account and see every Loan Application tied to it grouped together, instead of one flat list — a small, concrete instance of relationship-centric data unification.
- Add an AI-assisted "recommended next action" per application, generated by calling the Anthropic API directly (not Salesforce Agentforce, which isn't licensed here) based on the application's current Status and how long it's been there.
- Enable Salesforce Field History Tracking on `Loan_Application__c.Status__c` and surface the real resulting audit trail (`Loan_Application__History`) as a status-change timeline per application — genuine Salesforce audit data, not a derived/fake timeline — explicitly paralleled in copy to the existing Farpost "computed from events, not a stored score" framing already on the page.
- All three additions are read-only against Salesforce — no new public write surface, consistent with the existing decision to keep write access to Create and a scoped Status-only Update.

## Capabilities

### New Capabilities
- `salesforce-relationship-view`: the Account-grouped view, the AI next-action suggestion endpoint/UI, and the Field History Tracking-backed timeline endpoint/UI, plus the portfolio page copy tying the timeline back to the Farpost parallel.

### Modified Capabilities
(none — `salesforce-loan-demo` hasn't been archived into `openspec/specs/` yet, so this is scoped as purely additive rather than a formal delta against it)

## Impact
